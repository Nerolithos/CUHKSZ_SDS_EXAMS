# CUHKSZ papers for SDS students

中文：

這裏收集了2020年以來儘量多的 SDS 各科考卷（主要是 Major Mandatory），以供各位龍大校友考試複習參考，祝好。

請注意以我的經驗，刷題對於 PHY1001\DDA2001 和 STA2001 尤其有幫助，但對於 CSC3200、CSC3001 和 MAT2041 可能沒那麼有效，你還是應當以理解概念爲主。

你被允許將這些考卷傳到 Studocu 或者 Coursehero 上，如果你願意。

但請不要將它們用作你作弊的資料，因爲我校對作弊的打擊十足嚴格。一切後果責任由你自己承擔。



English:

This collection contains as many SDS exam papers as possible since 2020, mainly for major mandatory courses, for fellow Loongers to use as revision materials for exams. Best of luck.

Please note that, in my experience, practicing past papers is especially helpful for PHY1001, DDA2001, and STA2001, but may be less effective for CSC3200, CSC3001, and MAT2041, for which you should focus more on understanding the concepts.

You are allowed to upload these exam papers to Studocu or Course Hero if you wish.

However, please do not use them for cheating, as our university very strict in dealing with academic dishonesty. You alone are responsible for any consequences.



Bahasa Indonesia:

Di sini dikumpulkan sebanyak mungkin soal ujian SDS sejak tahun 2020, terutama untuk mata kuliah wajib jurusan, sebagai bahan referensi persiapan ujian bagi sesama alumni Loong. Semoga sukses.

Harap diperhatikan bahwa, berdasarkan pengalaman saya, mengerjakan soal-soal latihan sangat membantu khususnya untuk PHY1001, DDA2001, dan STA2001, tetapi mungkin tidak terlalu efektif untuk CSC3200, CSC3001, dan MAT2041. Untuk mata kuliah tersebut, kamu tetap sebaiknya lebih berfokus pada pemahaman konsep.

Kamu diperbolehkan mengunggah soal-soal ujian ini ke Studocu atau Course Hero jika kamu mau.

Namun, mohon jangan menggunakan materi ini untuk menyontek, karena universitas kita sangat ketat dalam menindak kecurangan akademik. Segala konsekuensi menjadi tanggung jawabmu sendiri.
